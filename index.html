PFMEA Quick Guide

Severity (SEV) — How serious is the effect? 1=none, 10=hazardous.

Occurrence (OCC) — How likely is failure? 1=almost never, 10=almost certain.

Detection (DET) — Can controls detect it? 1=almost certain, 10=cannot detect.

RPN = SEV × OCC × DET. Range 1–1000.

🔴 RPN ≥ 200 — Immediate action required.
🟡 RPN 100–199 — Action recommended.
🟢 RPN < 100 — Monitor periodically.

Chart View

Switch to ⬡ Chart to see the full hierarchy visually.

Click any node to open its PFMEA on the right.

Right-click a node to add children, rename, or delete.

Coloured dots on nodes: 🔴 High RPN, 🟡 Medium, 🟢 Low.

⊡ Fit resets the scroll to the top-left.

How to Use

1. Click + Add and choose Process to add a top-level process.

2. Click + Add → Component and choose which process to add it under.

3. Click + Add → Sub-component and choose which component to add it under.

4. Click any node to open its PFMEA table.

5. RPN calculates automatically as you type.

6. Export to save; Import to reload on another machine.
